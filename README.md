# Forest Fire Recognition System Using Machine Learning Models

## Project Overview
This project detects forest fires using Machine Learning and Image Processing techniques. 
The system analyzes forest images and classifies them as Fire or No Fire.

## Technologies Used
- Python
- OpenCV
- TensorFlow / Keras
- NumPy
- Matplotlib

## Features
- Detect forest fire from images
- Image classification (Fire / No Fire)
- Early fire detection system

## Project Structure
app.py – Main application file  
data/ – Dataset files  
templates/ – HTML templates  
static/ – Images and uploads  
utils/ – Helper functions  

## How to Run the Project

1 Install required libraries

pip install -r requirements.txt

2 Run the application

python app.py

## Note
The trained model file is not included in this repository because it exceeds GitHub size limits.

Place the trained model inside:

models/fire_detection_model.h5
