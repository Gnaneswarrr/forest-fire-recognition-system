How It Works
1. Satellite Detection
NASA's VIIRS sensor aboard the Suomi NPP satellite scans Earth's surface, detecting thermal anomalies (fires) via infrared imaging.
2. FIRMS Processing
NASA's FIRMS system processes raw satellite data within 3 hours, generating fire hotspot coordinates with confidence levels and fire intensity (FRP).
3. Real-Time Mapping
Our system pulls this live data and plots each fire hotspot on the map — showing location, intensity, and detection time.
